# n8n Workflow

This folder contains the reconstructed n8n workflow used in the project.

## Workflow Components

### Trigger
- WhatsApp Cloud API Trigger

### Text Branch
- JavaScript preprocessing
- AI Agent
- Google Gemini Chat Model
- Simple Memory
- WhatsApp Reply

### Image Branch
- Download WhatsApp Media
- Base64 Conversion
- Gemini Vision API
- Response Formatting
- WhatsApp Reply

## Note

The original workflow export was lost after the n8n Cloud trial expired before an export could be created.

The `workflow.json` included in this repository is a reconstruction based on:

- Final workflow screenshots
- Project documentation
- Tested implementation
- Development notes
- Actual workflow architecture

The reconstructed workflow accurately represents the implemented logic and is intended to preserve the project for learning and portfolio purposes.
